> 所谓“讲信誉”或叫诚信或本分都是在需要付出代价的时候才能体现出来。本分的意思就是1.0的厚度就说是1.0的厚度，句号。

2026-W17 和 2026-W18，这两周的周报也放一起写。

做了什么事：
- 调整个人博客架构
- 调整日语句子卡片
- 构建新类型 Anki 卡片
- 调整个人 GitHub 
- 整理个人磁盘的数据，包括电子书、影视内容和写真照片
- 放弃几个日常项目，包括 Telegram GitHub Issue Bot、GitHub Ebook Reader

上面的事情都是通过 AI CLI 完成的。当然，看了看这两周的记录，没有做什么可能比做了什么更重要。比方说，我竟然考虑再买一台笔记本电脑来上下班通勤使用。实际上，这个决策毫无道理可言，因为我可以带着当前这台笔记本上下班。许多事情当时看起来合情合理，可换个时间点换个环境、想远一点看，就没有道理可言。没有做的事情就不列了，太多了。还是不够理性。

这两周主要是想明白一些事情，想明白以后，就自然放弃某些事情。比方说，通过 AI，我已经能够实现：通过 Issue 来写东西，而后通过 GitHub Action 来自动创建并推送到当前仓库的 `_post` 并生成博客文章。那么，我有必要再去本地 Vim 编辑 md 文件吗？没有必要的。再比如说，Telegram GitHub Issue Bot，其功能是通过 Telegram Bot 将信息通过 Bot 保存到个人笔记仓库的 Issues 中：可这不是脱裤子放屁吗？在能够访问 GitHub 的情况下，可以直接在对应的 Issue 中编辑文字，何苦再通过 Telegram Bot 来中转呢？添加一个 comment 没有那么麻烦。那为什么这个项目运行了好几个月、现在才意识到这是脱裤子放屁呢？还是思维惯性、没有弄清楚个人的实际需求。

还做过一些错事。阿段说的是对的，有些事情，你短期考虑是很难去决定做还是不做，可你从十年的角度去考虑，那么做还是不做就清晰得多。当然，我不能说做过什么错事、做错了什么事。

AI 确实很强。关于 AI，我想起荀子的一段话：
> 吾嘗終日而思矣，不如須臾之所學也。吾嘗跂而望矣，不如登高之博見也。登高而招，臂非加長也，而見者遠；順風而呼，聲非加疾也，而聞者彰。假輿馬者，非利足也，而致千里；假舟楫者，非能水也，而絕江河。君子生非異也，善假於物也。

汽车不仅仅是驾驶工具，还是规模杀伤性武器。正因此，社会才衍生出「驾驶证」的要求。「驾驶证」降低的是开车技术的风险，没办法降低人主观杀人的意愿。那么，AI 呢？不管怎么说，我打算整一台可以运行本地 LLM 的笔记本。
